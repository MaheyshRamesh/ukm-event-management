# Senarai Semak Bendahari (Bendahari Checklist)

A step-by-step checklist for the program bendahari. Copy this into your own project tracking doc and tick off as you go.

---

## Phase 1: Pre-Program Planning (T-8 to T-6 weeks)

- [ ] Kertas Kerja drafted with detailed belanjawan
- [ ] Program registered in **iSTAR** system
- [ ] iSTAR approval obtained (Surat Kelulusan iSTAR)
- [ ] Funding source(s) identified (HEP-UKM / Faculty / KMUKM / Sponsorship)
- [ ] **Borang Permohonan Dana** submitted to relevant body
- [ ] **Borang Aku Janji** signed (if KMUKM funding)
- [ ] Pitching session attended (if KMUKM dana > threshold)

## Phase 2: Funding & Sponsorship (T-6 to T-4 weeks)

- [ ] Dana application approved — **Surat Kelulusan Dana** received
- [ ] If seeking external sponsors: **Borang Permohonan Memungut Derma** submitted to HEP-UKM (2 weeks before program)
- [ ] Sponsorship kit / prospectus sent to potential sponsors
- [ ] Tax exemption letter (**Surat Pengecualian Cukai**) prepared if sponsors need it
- [ ] Sponsorship commitments confirmed and logged

## Phase 3: Procurement & Logistics (T-4 to T-2 weeks)

### Venue
- [ ] Venue identified (AST / KPZ / other)
- [ ] Contacted venue coordinator (e.g., En. Salleh for AST)
- [ ] Program details emailed to venue
- [ ] iSTAR approval confirmed (required before SMPR booking)
- [ ] Booking completed via **SMPR** (https://ewarga.ukm.my/smpr/index.cfm)

### Jamuan (Food & Beverage)
- [ ] Total jamuan amount calculated
- [ ] If < RM500: **Borang Pesanan Jamuan** (Borang Pink / UKM-SPKPPP-PP04-BO02) filled and submitted with PTj signature
- [ ] If ≥ RM500: **Pesanan Rasmi** route initiated (see below)
- [ ] Jamuan rates verified against Pekeliling Bendahari Bil. 8/2022 + Pindaan Bil. 4/2025

### Pesanan Rasmi (for any purchase ≥ RM500)
- [ ] 5 documents prepared: Surat Permohonan, Kertas Kerja iSTAR, Surat Kelulusan iSTAR, Surat Kelulusan Dana, Spesifikasi + anggaran harga
- [ ] Hardcopy submitted to Kewangan HEP-UKM (**minimum 2 weeks before event**)
- [ ] eP@UKM broadcast completed by HEP-UKM (1-3 business days)
- [ ] Quotations received from vendors via HEP-UKM email
- [ ] Vendor selected — replied to HEP-UKM with choice + justification
- [ ] Pesanan Rasmi issued
- [ ] If UKM logo on any item: confirmed vendor is Pembekal Berlesen Cop Dagangan UKM

### Pendahuluan (Cash Advance for items < RM500)
- [ ] **Borang Permohonan Pendahuluan Kegiatan Pelajar** filled
- [ ] All required documents attached (kelulusan iSTAR, kelulusan dana, kertas kerja, belanjawan)
- [ ] Submitted to UKPEL (min 2 weeks before event)
- [ ] Previous pendahuluan fully settled (required for new approval)

### Equipment
- [ ] **Borang Permohonan Peralatan & Kemudahan Majlis** submitted if needed

## Phase 4: Event Day

- [ ] All receipts collected and dated correctly (during/before program period)
- [ ] Attendance lists filled for jamuan (senarai nama peserta)
- [ ] Photos taken for documentation (if needed for report)
- [ ] Keep ALL original receipts safe — you need hardcopy originals

## Phase 5: Post-Program (T+1 to T+30 days)

- [ ] Determine report type: Pendahuluan (30-day deadline) or Tuntutan (60-day deadline)
- [ ] **Format Surat Serahan Laporan Kewangan** downloaded from iSTAR
- [ ] **Penyata Kewangan** (financial statement) prepared
- [ ] All receipts organized and attached (hardcopy originals)
- [ ] Report signed by Dekan / Penasihat as required
- [ ] **Pengakuan Penyerahan** form signed
- [ ] Report submitted to UKPEL / Portal Pelajar Bendahari
- [ ] Photocopy of entire report kept for your records
- [ ] If surplus/remaining Pendahuluan: return excess to UKM

## Quick Reference: Key Deadlines

| Deadline | What |
|----------|------|
| T-8 weeks | Start Kertas Kerja & iSTAR registration |
| T-6 weeks | Submit dana application |
| T-4 weeks | Begin venue booking, sponsorship outreach |
| T-2 weeks | **Hard deadline:** Pesanan Rasmi & Pendahuluan submission |
| T-day | Event |
| T+30 days | Laporan Kewangan (Pendahuluan) deadline |
| T+60 days | Laporan Kewangan (Tuntutan) deadline |
