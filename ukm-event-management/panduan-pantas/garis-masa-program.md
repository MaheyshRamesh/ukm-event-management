## Week-by-Week Breakdown

### T-8 weeks (8 weeks before event)
| Task | Owner | Notes |
|------|-------|-------|
| Draft Kertas Kerja with detailed belanjawan | Setiausaha + Bendahari | Include all dana sources in budget |
| Register program in iSTAR | Ketua Program | Cannot proceed without this |
| Identify venue requirements | Logistik | Contact venue coordinator early |

### T-6 weeks
| Task | Owner | Notes |
|------|-------|-------|
| Submit dana application (Faculty / KMUKM / HEP) | Bendahari | Different bodies = different forms |
| Submit Borang Aku Janji (if KMUKM) | Bendahari | Signed by Penasihat KMUKM |
| Begin sponsor outreach | Pemasaran | Send kit penajaan to targets |

### T-4 weeks
| Task | Owner | Notes |
|------|-------|-------|
| Confirm dana approval (Surat Kelulusan Dana) | Bendahari | Follow up if no response |
| Finalize venue booking (SMPR) | Logistik | iSTAR approval required first |
| Submit Borang Permohonan Memungut Derma | Bendahari | If accepting external sponsorship (2 weeks before program rule) |
| Identify all items needing Pesanan Rasmi (≥ RM500) | Bendahari | List everything: jamuan, printing, packages |

### T-2 weeks ⚠️ HARD DEADLINE
| Task | Owner | Notes |
|------|-------|-------|
| **Submit Pesanan Rasmi documents to HEP-UKM** | Bendahari | 5 documents, hardcopy, cannot be late |
| **Submit Pendahuluan application to UKPEL** | Bendahari | All supporting docs attached |
| Submit Borang Pesanan Jamuan (Borang Pink) | Bendahari | For jamuan < RM500 |
| Submit Borang Peralatan & Kemudahan Majlis | Logistik | If borrowing UKM equipment |

### T-1 week
| Task | Owner | Notes |
|------|-------|-------|
| Confirm Pesanan Rasmi vendor selection | Bendahari | Reply to HEP-UKM email with justification |
| Confirm Pendahuluan cash received | Bendahari | Check bank account |
| Confirm all vendor delivery dates | Bendahari + Logistik | Especially jamuan timing |
| Prepare receipt collection system | Bendahari | Envelopes/folders for each spending category |

### T-Day (Event)
| Task | Owner | Notes |
|------|-------|-------|
| Collect ALL receipts | Bendahari | Original hardcopy, dated correctly |
| Fill attendance lists for jamuan | Jamuan AJK | Must match pax in Borang Pesanan |
| Document everything | Media | Photos useful for Laporan Kewangan |

### T+1 to T+30 days
| Task | Owner | Notes |
|------|-------|-------|
| Prepare Penyata Kewangan | Bendahari | Categorize all income & expenses |
| Compile all receipts (hardcopy originals) | Bendahari | Organized by category |
| Get signatures (Dekan/Penasihat) | Bendahari | On Surat Serahan and report |
| **Submit Laporan Kewangan (Pendahuluan)** | Bendahari | 30-day hard deadline |
| Sign Pengakuan Penyerahan | Bendahari | Keep a photocopy |

### T+31 to T+60 days (if claiming Tuntutan only)
| Task | Owner | Notes |
|------|-------|-------|
| **Submit Laporan Kewangan (Tuntutan)** | Bendahari | 60-day hard deadline |
| Return any excess Pendahuluan | Bendahari | If applicable |
