# Faculty-Level Funding (Dana Fakulti)

Each faculty has its own process for approving student program funding. This folder includes FTSM's procedure as a concrete example — check with your own faculty's HEP office for specifics.

---

## A-to-Z Flow: Faculty Funding (FTSM Example)

```mermaid
flowchart TD
    A["1. Prepare Kertas Kerja Fakulti\n(include detailed Belanjawan per\nfunding source)"] --> B["2. Submit to Mesyuarat HEPA\n& Mesyuarat Pengurusan\nFakulti (MPF)"]
    B --> C["3. MPF reviews & decides\nfunding amount"]
    C --> D["4. Receive Petikan Minit\nMesyuarat Fakulti"]
    D --> E{"Need Pendahuluan\n(advance cash)?"}
    E -->|"Yes"| F["5. Prepare 5 documents:\n• Borang Pendahuluan Kegiatan Pelajar\n• Surat Kelulusan iSTAR\n• Petikan Minit MPF\n• Kertas Kerja iSTAR\n• Belanjawan Terperinci"]
    E -->|"No"| H["Skip to program execution"]
    F --> G["6. Dekan FTSM signs as\nKetua Tabung"]
    G --> G2["7. Combine all docs into 1 PDF\n& upload via Google Form\n(Portal Pelajar Bendahari)"]
    G2 --> G3["8. Receive email notification\nwhen Pendahuluan is approved"]
    G3 --> H
    H --> I["9. Run program"]
    I --> J["10. Submit Laporan Kewangan\n(signed by Dekan) via\nPortal Pelajar Bendahari\nwithin 30 days"]
```
