## [2026-03-20] — Initial Release

### Added
- Complete repository structure with 7 main sections + cheatsheets
- 28 source documents (PDFs, XLSX, DOCX) organized from scattered files
- Root README with program lifecycle flowchart and spending decision tree
- `CLAUDE.md` for Claude Code integration
- `PLAN.md` documenting all repo decisions and structure
- `CONTRIBUTING.md` with maintenance guide for future sessions

### Section READMEs
- `01-garis-panduan/` — Summary of Pekeliling Bendahari Bil. 8/2022 + Pindaan Bil. 4/2025
- `02-sumber-dana/` — Funding decision tree + sub-guides for HEP, Faculty, KMUKM, Penajaan
- `03-belanjawan/` — Budget preparation guide with templates
- `04-perbelanjaan/` — Master spending decision tree + sub-guides for all 5 spending channels (Pendahuluan, Pesanan Rasmi, Jamuan, Bayaran Terus, Caj Antara Jabatan)
- `05-tempat-dan-logistik/` — Venue booking guides for AST and kolej kediaman
- `06-laporan-kewangan/` — Financial reporting guide with flowchart and rejection reasons
- `07-latihan/` — Training materials reference with suggested training agenda

### Cheatsheets
- `panduan-pantas/senarai-semak-bendahari.md` — Full bendahari checklist (checkbox format)
- `panduan-pantas/garis-masa-program.md` — T-8 weeks countdown timeline with Gantt chart
- `panduan-pantas/glosari.md` — Glossary of UKM-specific terms, systems, and thresholds
- `04-perbelanjaan/jamuan/kadar-jamuan.md` — Jamuan rate table and eligibility rules

### Automation
- `scripts/scrape-public.py` — Public UKM page scraper
- `scripts/check-pekeliling.py` — Pekeliling update checker
- `scripts/auto-update.py` — Orchestrator for autonomous updates
