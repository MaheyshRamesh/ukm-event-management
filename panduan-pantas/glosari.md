# Glosari (Glossary)

A reference for UKM-specific acronyms, terms, and systems that appear throughout this repo.

---

## Systems & Platforms

| Term | Full Name | What It Is |
|------|-----------|------------|
| **iSTAR** | Integrated Student Activity Record | UKM's central system for registering, approving, and tracking student programs. Every program must be registered here before anything else can happen. |
| **eP@UKM** | e-Perolehan UKM | UKM's electronic procurement system (https://ukmperolehan.ukm.my/). Used for Pesanan Rasmi — broadcasts purchase requirements to registered vendors. |
| **SMPR** | Sistem Maklumat Pengurusan Ruang | UKM's room/venue booking system (https://ewarga.ukm.my/smpr/index.cfm). Requires login. Search by activity keyword (min 3 characters). |
| **iFASt** | Integrated Financial Accounting System | UKM's internal billing system for Caj Antara Jabatan (inter-department charges). Used when one UKM unit uses services from another. |
| **Portal Pelajar Bendahari** | — | Online portal for students to submit pendahuluan applications and laporan kewangan (https://bendahari.ukm.my/student_portal). |

## Organizations & Units

| Term | Full Name | What It Is |
|------|-----------|------------|
| **HEP-UKM** | Hal Ehwal Pelajar UKM | Student Affairs Division at university level. Handles program approvals, Pesanan Rasmi processing, and is the main interface for student financial matters. |
| **HEP FTSM** | Hal Ehwal Pelajar FTSM | Faculty-level student affairs office at FTSM. Different faculties have their own HEP. |
| **KMUKM** | Kesatuan Mahasiswa UKM | UKM Student Union. One of the main funding sources for student programs. |
| **UKPEL** | Unit Kewangan Pelajar, Jabatan Bendahari | Student Finance Unit within the Treasurer's Office. Handles pendahuluan approvals, payment processing, and laporan kewangan. Contact: 03-8911 8178/8599/8182. |
| **PTj** | Pusat Tanggungjawab | Responsibility Center — the administrative unit that oversees and approves financial matters (e.g., your faculty, HEP-UKM, KMUKM). |
| **IVI** | Institut Visual Informatik | Institute that manages AST (Aras Serbaguna Tunku) facilities. |

## Financial Terms

| Term | What It Means |
|------|---------------|
| **Pendahuluan** | Cash advance disbursed to organizer's bank account before the program. Must submit Laporan Kewangan within 30 days after event. |
| **Tuntutan Bayaran Balik** | Reimbursement claim submitted after paying out of pocket. 60-day deadline. |
| **Pesanan Rasmi** | Official Purchase Order — required for any purchase ≥ RM500. Goes through eP@UKM. Must submit 2 weeks before event. |
| **Borang Pink** | Borang Pesanan Jamuan (UKM-SPKPPP-PP04-BO02) — the pink-colored form used to order food/drinks for in-campus events costing < RM500. Picked up from HEP-UKM office. |
| **Bayaran Terus** | Direct Payment — payment made directly through UKM's account to speakers, judges, prize winners, or external institutions. |
| **Caj Antara Jabatan** | Inter-department charge via iFASt — used when your program uses services from another UKM unit (room rental, equipment, printing, etc.). |
| **Laporan Kewangan** | Financial report submitted after the program. Two types: Pendahuluan (30-day deadline) and Tuntutan (60-day deadline). |
| **Penyata Kewangan** | Financial statement — the income vs. expenditure summary within the Laporan Kewangan. |
| **Saguhati** | Honorarium/token payment for speakers, facilitators, judges. |

## Documents & Forms

| Term | Full Name | When It's Used |
|------|-----------|----------------|
| **Kertas Kerja** | Program proposal/paper | Submitted during iSTAR registration, included in most applications |
| **Surat Kelulusan iSTAR** | iSTAR approval letter | Required for virtually every financial process |
| **Surat Kelulusan Dana** | Funding approval letter | Proof that your funding has been approved |
| **Borang Aku Janji** | Pledge form | Signed commitment to KMUKM rules when receiving KMUKM funding |
| **Borang Permohonan Memungut Derma** | Donation collection application form | Required when accepting external sponsorship (submit 2 weeks before program) |
| **Surat Pengecualian Cukai** | Tax exemption letter | Given to sponsors so they can claim tax deduction for supporting UKM activities |
| **Pengakuan Penyerahan** | Submission acknowledgment | Signed declaration when submitting Laporan Kewangan |
| **Belanjawan Terperinci** | Detailed budget | Line-item budget required in Kertas Kerja and dana applications |

## Key Thresholds

| Number | Significance |
|--------|-------------|
| **RM500** | Below = Borang Pink / Pendahuluan. At or above = Pesanan Rasmi |
| **RM3,000** | Above = Pesanan Rasmi handled by Bahagian Perolehan, Jabatan Bendahari (not HEP) |
| **2 weeks** | Minimum lead time before event for Pesanan Rasmi and Pendahuluan |
| **30 days** | Laporan Kewangan (Pendahuluan) deadline after event |
| **60 days** | Laporan Kewangan (Tuntutan) deadline after event |
| **14 working days** | Payment processing time by Jabatan Bendahari |
| **6 hours / 12:30pm** | Jamuan eligibility thresholds (Pindaan Bil. 4/2025) |
| **1–3 working days** | eP@UKM broadcast duration for vendor bidding |
