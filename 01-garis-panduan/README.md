## Policy Summary

### Pekeliling Bendahari Bil. 8/2022 (Original)

Sets the official rate ceilings for:
- University jamuan (lunch, light refreshments, dinner)
- Hotel package eligibility
- Hospitality for visiting lecturers/professors

### Amendment: Pindaan Bil. 4/2025 (Effective 13 June 2025)

The key change — jamuan eligibility was **relaxed**:

| Before (Bil. 8/2022) | After (Pindaan Bil. 4/2025) |
|------------------------|-------------------------------|
| Lunch only for programs > 6 hours | Programs > 6 hours → Jamuan Ringan + Lunch |
| Programs < 6 hours → light refreshments only | Programs < 6 hours **but crossing 12:30pm** → Jamuan Ringan / Lunch allowed |
| — | Events with LPU Chair / LPU Members / VC / external guests → Jamuan Ringan + Lunch |
| Meetings < 6 hours → light refreshments only | This clause is **CANCELLED** |

> **Practical impact:** A 4-hour workshop from 10am–2pm now qualifies for lunch because it crosses 12:30pm. This gives organizers significantly more flexibility for short programs.

### Current Form Version

Following the amendment, the required jamuan form is **UKM-SPKPPP-PP04-BO02** (Effective Date: 06 Jun 2025). Older versions are **no longer valid**.
